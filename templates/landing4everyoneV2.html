1강. 노션 페이스로 프로필 사진 만들기

이미지는 페이지를 채워

노션 페이스를 활용하여 멋진 프로필 사진을 만드는 방법을 배워보세요.

  1. https://faces.notion.com/로 접속하기
  2. 새로운 얼굴 버튼을 눌러서 새 프로필 이미지 만들기
  3. 또는 랜덤 버튼을 눌러서 랜덤 프로필 이미지 만들기
  4. 다음 버튼 눌러서 생성한 이미지 다운로드하기 (파일 이름은 face.PNG로 저장)
  5. 특정 경로에 옮겨 저장하기 (예: C:\Users\사용자이름\Desktop\landing\face.PNG)

2강. GitHub Codespace 설정하기

제공된 코드를 복사하여 GitHub Codespace에 붙여넣고, 자신의 정보로 수정한 후 GitHub Pages로 배포하는 방법을 배웁니다.

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>나만의 랜딩 페이지</title>
</head>
<body>
    <header>
        <h1>나만의 랜딩 페이지</h1>
        <img src="face.PNG" alt="프로필 이미지">
    </header>
    <main>
        <p>이곳에 자신을 소개하는 내용을 작성하세요.</p>
    </main>
</body>
</html>

위 코드를 복사하여 GitHub Codespace에 붙여넣고, 자신의 정보로 수정하세요.

3강. HTML 설명하기

HTML은 웹페이지를 만드는 가장 기본적인 언어입니다. 마치 집을 지을 때의 설계도와 같아요!

기본 구조

  • <!DOCTYPE html> - "이 문서는 HTML입니다!"라고 알려주는 문장
  • <html> - 웹페이지의 시작과 끝을 나타내는 태그
  • <head> - 웹페이지의 정보를 담는 부분
  • <body> - 실제로 보여지는 내용을 담는 부분

4강. CSS 설명하기

CSS는 웹페이지의 디자인을 담당하는 언어입니다. HTML이 집의 구조라면, CSS는 인테리어와 같아요!

기본 사용법

  • 선택자 - 스타일을 적용할 요소를 선택
  • 속성 - 어떤 스타일을 적용할지 지정
  • 값 - 속성에 적용할 구체적인 값

5강. JavaScript 설명하기

JavaScript는 웹페이지에 동적인 기능을 추가하는 프로그래밍 언어입니다. 웹페이지에 생명을 불어넣는 역할을 합니다!

기본 기능

  • 이벤트 처리 - 버튼 클릭, 마우스 이동 등의 이벤트에 반응
  • DOM 조작 - 웹페이지의 요소를 동적으로 변경
  • 데이터 처리 - 서버와 통신하여 데이터를 주고받음
